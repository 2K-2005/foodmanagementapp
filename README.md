# 🍔 Food Ordering Application (Swiggy Clone)

A full-stack Food Ordering Application inspired by Swiggy, developed using Java and modern web technologies. This project demonstrates end-to-end development including backend APIs, frontend UI, and database integration.

---

## 🚀 Features

- 🍽️ Browse food items
- 🛒 Add items to cart
- 📦 Place and manage orders
- 🔄 Dynamic UI rendering
- 🔗 Seamless frontend-backend integration
- 🗂️ CRUD operations for orders and data management

---

## 🛠️ Tech Stack

### 🔹 Backend
- Java
- Spring Boot
- Spring Data JPA
- REST API

### 🔹 Frontend
- React.js
- HTML5, CSS3, JavaScript

### 🔹 Database
- MySQL

### 🔹 Tools
- Postman (API Testing)
- Git & GitHub
- Maven

---

## 🏗️ Architecture

The application follows a layered architecture:

Controller → Service → Repository → Database

- **Controller Layer**: Handles HTTP requests and API endpoints  
- **Service Layer**: Contains business logic  
- **Repository Layer**: Manages database operations  

---

## 📌 Key Functionalities

- Developed RESTful APIs for order management and data handling  
- Integrated React frontend with backend APIs using HTTP requests  
- Implemented state management using React Hooks (useState, useEffect)  
- Managed data exchange using JSON format  
- Ensured modular and maintainable code structure
